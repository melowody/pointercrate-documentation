# Named ID{id=named-id}

Object representing some generic name-id pair.

| Field        | Type   | Description                                                     |
| ------------ | ------ | --------------------------------------------------------------- |
| id   | integer | The ID associated with the object |
| name | string? | The name associated with the object | 

## Example object

```json
{
    "id": 5,
    "name": "cat"
}
```
